This repository contains the "Coming Soon" landing page for Noto, showcasing its features and letting visitors sign up for launch notifications.

- Brain Dump & AI Timeline
- Cortex AI (OpenAI + Ollama)
- Voice Notes with Whisper transcription
- Universal Search
- 8 Themes
- Cross-platform (iOS, Android, macOS, Windows, Linux)
